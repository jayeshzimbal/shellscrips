#!/bin/bash

#! is called a shebang.
It tells Linux which interpreter should execute the script.
/bin/bash means use the Bash (Bourne Again Shell) interpreter.
Bash is an advanced shell with many extra features.
Supports:
Arrays
Functions
Command history
Tab completion
Arithmetic operations
Advanced scripting features


#!/bin/sh

#! is the shebang.

It is simpler and more portable than Bash.
Supports basic shell scripting features.
On many Linux systems, /bin/sh is a link to another shell (such as dash or bash).


#!/bin/bash = Powerful shell for scripting and automation
#!/bin/sh = Simple, portable shell for basic scripts
